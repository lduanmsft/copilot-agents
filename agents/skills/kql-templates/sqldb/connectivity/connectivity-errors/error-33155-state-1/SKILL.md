## Description:
33155 is usually a client side issue
Clients failing Active Directory logins to SQL DB because the client is not able to grab an authentication token from Active Directory (AD) or Azure Active Directory (AAD).
A disconnect event was raised when server is waiting for Federated Authentication token. This could be due to client close or server timeout expired.
