5 Frontend Plugins

Plugins 在 Bot Framework 前端层运行，负责 UI 交互和辅助功能：

Control Plugins

  • Control.Suggestions — 对话启动推荐
  • Control.ChatContextReset — 会话重置
  • Control.QuestionRecommendations — 后续问题推荐
  • Control.Notifications — 推送通知

Functional Plugins

  • Kql.KqlHandler — Kusto 查询生成与执行
  • OnCall.DailySummary — OnCall 每日 incident 汇总
